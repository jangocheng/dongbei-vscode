# dongbei-vscode

东北话编程vscode语法高亮插件

东北话编程主仓库： https://github.com/zhanyong-wan/dongbei

## 特性

亮就完事了。

## 已知问题

- [] 部分关键字高亮不够美型
- [] syntax awareness识别（例如识别`算九九乘法咋整：`中函数名为`算九九乘法`，而非高亮数值关键字`九九`）

## 测试方法

git clone最新版repo（确保`.vscode/launch.json`文件存在）后直接F5（或在debug面板中点击启动）启动插件。
